# Android Activities Course

This repository contains learning materials and exercises for an Android development course focused on **Activities** - one of the fundamental building blocks of Android applications.

## What You'll Learn

This course covers the essential concepts of Android Activities, including:

- **Activity Lifecycle** - Understanding onCreate(), onStart(), onResume(), onPause(), onStop(), and onDestroy()
- **Activity Navigation** - Moving between different screens using Intents
- **State Management** - Saving and restoring activity state across configuration changes
- **Activity Communication** - Passing data between activities
- **Activity Stack & Back Navigation** - Understanding the back stack and task management

## Prerequisites

- Basic knowledge of Kotlin or Java
- Android Studio installed
- Understanding of object-oriented programming concepts

## Getting Started

Each module in this course will contain:
- Sample code demonstrating key concepts
- Hands-on exercises to practice what you've learned
- Best practices for building robust Android applications

## Course Structure

Projects and exercises will be organized by topic, with each focusing on specific aspects of Android Activities.
